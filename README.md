# Email App

여러 이메일 계정을 하나의 앱에서 통합 관리하고, AI를 활용해 메일을 요약·검색하는 실습 프로젝트입니다.

## 주요 목표

- Gmail, 네이버 메일 등 여러 이메일 계정 연동
- 통합 받은편지함 제공
- 메일 상세 조회 및 검색
- AI 기반 메일 요약
- 자연어 기반 메일 검색
- 사용자 계정 및 메일 계정 보안 관리

## 기술 스택
